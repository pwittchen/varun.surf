## varun.surf

weather forecast and real-time wind conditions dashboard for kitesurfers

see it online at: https://varun.surf

## building

backend:

```
./gradlew clean bootJar
```

frontend:

```
./build_frontend.sh
```

## running

```
java --enable-preview -jar build/libs/*.jar
```

## testing

```
./gradlew test
```
