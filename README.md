## varun.surf

weather forecast and real-time wind conditions dashboard for kitesurfers

## building the project

```
./gradlew clean build
```

## building the executable jar file

```
./gradlew clean bootJar
```

## testing

```
./gradlew test
```

## running

```
java -jar build/libs/*.jar
```

or:

```
./gradlew bootJar
```
